# Token Swap DApp

A fully functional decentralized application that allows users to **swap between two custom ERC-20 tokens — TokenA and TokenB** on the **Sepolia Testnet**.

Built with **Solidity**, **Hardhat**, **React.js**, and **Ethers.js**.

🔗 **Live Demo**: [ayushvaishnav.com/project/p4](https://ayushvaishnav.com/project/p4)

---

## ⚙️ Features

- 🔐 Connect wallet via MetaMask  
- 🔄 Instantly swap TokenA ↔ TokenB  
- 📈 View real-time token balances  
- 🧠 Smart contracts deployed on Sepolia  
- 🖥️ Intuitive React-based frontend  

---

## 🧱 Tech Stack

| Layer          | Tools / Libraries                |
|----------------|----------------------------------|
| 💡 Smart Contract | Solidity, Hardhat, OpenZeppelin |
| 🖼️ Frontend       | React.js, Ethers.js              |
| 🌐 Blockchain     | Ethereum (Sepolia Testnet)       |
| 👛 Wallet         | MetaMask                        |
| ☁️ Deployment     | Infura or Alchemy               |
